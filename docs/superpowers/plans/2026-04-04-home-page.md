# Home Page — T-Hex Garage Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the Home page do marketplace com seed de dados, componentes de UI (Header, BottomNav, ListingCard, ListingGrid, CategoryFilter) e a página principal que busca listings ativos do banco e os exibe em um grid filtrado por categoria.

**Architecture:** O `page.tsx` é um Server Component que busca listings e categorias via Prisma e passa para `HomeFeed`, um Client Component que gerencia o estado de categoria selecionada e renderiza o feed filtrado. Todos os outros componentes de UI são presentacionais e estáticos.

**Tech Stack:** Next.js 14+ App Router, TypeScript strict, Prisma ORM (PostgreSQL/Neon), Tailwind CSS v4 (cores via `@theme` em `globals.css`), lucide-react (ícones), tsx (runner do seed)

  label: string
  icon: React.ReactNode
  highlight?: boolean
}

const navItems: NavItem[] = [
  { href: '/', label: 'Home', icon: <Home size={22} /> },
  { href: '/buscar', label: 'Buscar', icon: <Search size={22} /> },
  {
    href: '/create',
    label: 'Criar',
    icon: <PlusCircle size={26} />,
    highlight: true,

  { href: '/salvos', label: 'Salvos', icon: <Bookmark size={22} /> },
  { href: '/perfil', label: 'Perfil', icon: <User size={22} /> },

export function BottomNav() {
  const pathname = usePathname()

  return (
    <nav className="fixed bottom-0 left-0 right-0 z-40 bg-linen border-t border-teal-muted/30 pb-safe">
      <ul className="max-w-screen-lg mx-auto flex items-center justify-around px-2 py-1">
        {navItems.map((item) => {
          const isActive = pathname === item.href
          return (
            <li key={item.href}>
              <Link
                href={item.href}
                className={cn(
                  'flex flex-col items-center gap-0.5 px-3 py-2 rounded-xl transition-colors',
                  item.highlight
                    ? 'text-linen bg-teal rounded-2xl px-4 py-2'
                    : isActive
                      ? 'text-airforce'
                      : 'text-teal-muted hover:text-teal'
                )}
                aria-label={item.label}
              >
                {item.icon}
                {!item.highlight && (
                  <span className="text-[10px] font-medium">{item.label}</span>
                )}
              </Link>
            </li>
          )
        })}
      </ul>
    </nav>
  )
}

      : listings.filter((l) => l.category.slug === selectedCategory)

  return (
    <div className="space-y-4">
      <CategoryFilter
        categories={categories}
        selected={selectedCategory}
        onChange={setSelectedCategory}
      />
      <ListingGrid listings={filtered} />
    </div>
  )
}

            },
